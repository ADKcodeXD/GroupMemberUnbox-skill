## 第一步：通读并标注噪声（必须先做）

在提取任何信息之前，先通读全部内容，识别以下噪声类型：

**噪声类型定义：**
- **复读 (Parrot)**：短时间内多人重复发送几乎相同的文字/表情/梗图，跟风行为，标注 `[复读/跟风]`。
- **玩梗 (Meme)**：互联网流行梗、表情包、抽象话、阴阳怪气的反讽（如"绷不住了"、"6"、"笑死"、"典"、"乐"），标注 `[疑似玩梗]`。
- **自黑/反讽 (Self-deprecation)**：故意贬低自己来获取社交认同（如"我就是个废物"），标注 `[疑似自黑]`，可信度**自动降1级**。跟风自黑→可信度直接为1，丢弃。
- **群体表演 (Group Performance)**：跟风骂/跟风吹，看是主动发起还是跟风，标注 `[群体表演]`。
- **发疯/情绪爆发 (Rant)**：突发的极端化发言，与前后语气明显断裂，标注 `[发疯/情绪爆发]`，可信度最高为2。
- **夸大/极端化 (Hyperbole)**：有意夸张（如"我月入百万"），标注 `[夸大/极端化]`，可信度最高为2。
- **媒体解说 (Media Explanation)**：在 `[图片]`、`[聊天记录]` 或 `[视频]` 之后进行的背景描述，很可能是在讲转发的内容而非个人事实。标注 `[疑似媒体解说]`，除非明确指向本人，否则不予提取。

## ⚡ 核心规则：语境意识与包容性提取

1. **媒体语境意识 (Media Context Awareness)**：
   - 聊天记录中可能出现 `[图片]` 或 `[聊天记录]` 等占位符。
   - 如果目标人物在占位符后立即开始背景描述或八卦解释，优先判定为“他者事实”，不予作为本人事实提取。
2. **包容性事实提取 (Enriched Extraction)**：
   - 对于有趣的生活碎片（健身、骑车、喜欢的电影、同学姓名、有趣的过往经历），即使只提到一次，也请保留并标注为 `可信度:2 (侧写/存疑)`。
   - 目标：不要因为严格的“2条证据”规则漏掉那些能体现人物“活生生感”的小细节。
3. **区分“自我”与“他者”**：
   - 确保提取的信息确实发生在目标人物身上（如“我同学” vs “转发的某篇文中提到的人”）。
