**⚠️ 证据使用规则：**
你使用的证据库已经过交叉验证，采用1-5级量化可信度（1=单次证据，2=偶尔提到，3=中等频率，4=高度反复，5=基本坐实）。你必须：
- 引用证据时同步标注其**可信度等级（数字）**
- 基于可信度1-2的证据做出的结论，必须加注"（推测，证据有限）"
- 基于可信度3的证据，可以谨慎陈述，注明"（中等可信度）"
- 基于可信度4-5的证据的结论，可以直接陈述
- 证据库分为A类(基本事实)、B类(状态事实)、C类(回忆过往经历)、D类(行为模式与情绪反应)、E类(喜好与厌恶)——分析时要区分**稳定属性**、**当前状态**、**过往经历**、**行为本能**以及**审美取向**

这份报告的风格是：**像一把手术刀，精准切开这个人的外壳**。但精准 ≠ 瞎猜。没有证据支撑的犀利，只是废话。

## 输出格式：

### 一、表层与隐层的核心矛盾
列出 3-6 组最尖锐的矛盾。**每组矛盾必须展示推理过程：**

```
#### 矛盾 N：[标题]
**表层呈现**：TA 在群里表现出的形象
- 证据："原话" (时间戳) [可信度:X/5]

**隐层现实**：证据揭示的真实状态
- 证据："原话" (时间戳) [可信度:X/5]

**推理过程**：为什么这两者构成矛盾？有没有其他解释？
- 对立假设：有没有可能表层和隐层并不矛盾，只是语境不同？→ [分析]
- 最终判断：[解释为什么你认为这确实是一个矛盾]

**矛盾本质**：这揭示了什么心理机制？
```

### 二、性格解剖
1. **核心性格底色**：用 2-3 个词高度概括
   - 每个词必须有**2条以上原话证据**支撑
   - 如果只有1条证据，降级为"疑似特征"
2. **MBTI 推演**：给出最可能的类型，展示推理过程（引用A/D类证据）
3. **防御机制**：最常用的防御机制，必须引用D类扣分/吐槽处的具体反应
4. **情绪引爆点**：什么话题让 TA 失控？必须引用D类记录的“破防”或强烈反应源

### 三、潜意识层分析
1. **最在意的事**：从自黑内容的反面或E类极致喜好的事物中推断
2. **文化/阶层画像**：通过E类（书影音、消费偏好）分析其受教育程度、审美高度、以及所属的亚文化群体（如：精英主义、二次元、大众跟随者等）
3. **内在价值序列**：TA 把什么排在第一位？（金钱、情感、尊重、智性自由？）引用D/E类证据支持。
4. **核心恐惧**：驱动一切行为的底层恐惧
5. **渴望认同的方式**

### 四、危险点与阿喀琉斯之踵
- **心理弱点**：最容易被刺痛的点
- **行为风险**：当前生活/决策模式中的最大风险
- **认知盲区**：TA 最大的盲区在哪？
  - 必须附上"为什么你认为 TA 看不到这个盲区"的推理

### 五、毒舌锐评（3句话）
用最精准、最犀利的方式总结。但**每句话后面必须用括号标注关键证据来源**。

### 六、可靠性声明
- 本报告中哪些结论最可靠（有多条高置信度证据支撑）？
- 哪些结论是推测性的（基于中/低置信度证据）？
- 如果你必须选出1个"最可能是错的"结论，是哪个？为什么？
